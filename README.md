<details open>
<summary>PyTorch Hub ile Çıkarım</summary>

Çıkarım için PyTorch Hub aracılığıyla YOLOv5'i kullanın. Modeller en son YOLOv5 sürümünden otomatik olarak indirilir.

Python

import torch

# Bir YOLOv5 modeli yükleyin (seçenekler: yolov5n, yolov5s, yolov5m, yolov5l, yolov5x)
model = torch.hub.load("ultralytics/yolov5", "yolov5s")  # Varsayılan: yolov5s

# Giriş görüntü kaynağını tanımlayın (URL, yerel dosya, PIL görüntüsü, OpenCV çerçevesi, numpy dizisi veya liste)
img = "https://ultralytics.com/images/zidane.jpg"  # Örnek görüntü

# Çıkarım yapın (otomatik olarak toplu işleme, yeniden boyutlandırma, normalleştirmeyi yapar)
results = model(img)

# Sonuçları işleyin (seçenekler: .print(), .show(), .save(), .crop(), .pandas())
results.print()  # Sonuçları konsola yazdırın
results.show()  # Sonuçları bir pencerede görüntüleyin
results.save()  # Sonuçları runs/detect/exp adresine kaydedin
</details>

<details>
<summary>detect.py ile Çıkarım</summary>

detect.py betiği çeşitli kaynaklarda çıkarım yapar. En son YOLOv5 sürümünden modelleri otomatik olarak indirir ve sonuçları runs/detect dizinine kaydeder.

Bash

# Bir web kamerası kullanarak çıkarım yapın
python detect.py --weights yolov5s.pt --source 0

# Yerel bir görüntü dosyasında çıkarım yapın
python detect.py --weights yolov5s.pt --source img.jpg

# Yerel bir video dosyasında çıkarım yapın
python detect.py --weights yolov5s.pt --source vid.mp4

# Ekran yakalama üzerinde çıkarım yapın
python detect.py --weights yolov5s.pt --source screen

# Bir görüntü dizininde çıkarım yapın
python detect.py --weights yolov5s.pt --source path/to/images/

# Görüntü yollarını listeleyen bir metin dosyasında çıkarım yapın
python detect.py --weights yolov5s.pt --source list.txt

# Akış URL'lerini listeleyen bir metin dosyasında çıkarım yapın
python detect.py --weights yolov5s.pt --source list.streams

# Görüntüler için bir glob deseni kullanarak çıkarım yapın
python detect.py --weights yolov5s.pt --source 'path/to/*.jpg'

# Bir YouTube video URL'sinde çıkarım yapın
python detect.py --weights yolov5s.pt --source 'https://youtu.be/LNwODJXcvt4'

# Bir RTSP, RTMP veya HTTP akışında çıkarım yapın
python detect.py --weights yolov5s.pt --source 'rtsp://example.com/media.mp4'

<summary>Eğitim</summary>

Aşağıdaki komutlar YOLOv5 COCO veri kümesi sonuçlarının nasıl yeniden üretileceğini gösterir. Hem modeller hem de veri kümeleri en son YOLOv5 sürümünden otomatik olarak indirilir. YOLOv5n/s/m/l/x için eğitim süreleri, tek bir NVIDIA V100 GPU'da yaklaşık 1/2/4/6/8 gündür. Çoklu GPU eğitimi kullanmak eğitim süresini önemli ölçüde azaltabilir. Donanımınızın izin verdiği en büyük --batch-size değerini kullanın veya YOLOv5 AutoBatch için --batch-size -1 kullanın. Aşağıda gösterilen toplu iş boyutları V100-16GB GPU'lar içindir.

Bash

# COCO üzerinde YOLOv5n'yi 300 epok boyunca eğitin
python train.py --data coco.yaml --epochs 300 --weights '' --cfg yolov5n.yaml --batch-size 128

# COCO üzerinde YOLOv5s'yi 300 epok boyunca eğitin
python train.py --data coco.yaml --epochs 300 --weights '' --cfg yolov5s.yaml --batch-size 64

# COCO üzerinde YOLOv5m'yi 300 epok boyunca eğitin

# COCO üzerinde YOLOv5x'i 300 epok boyunca eğitin
python train.py --data coco.yaml --epochs 300 --weights '' --cfg yolov5x.yaml --batch-size 16
